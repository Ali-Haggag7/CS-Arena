# مقدمة

مرحباً بك في توثيق **CS-Arena**. يغطي هذا الدليل كل ما تحتاج إلى معرفته — سواء كنت مستخدماً يعرض مشروعه، أو مطوراً يساهم في كتابة الكود.

## ما هي منصة CS-Arena؟

منصة CS-Arena مصممة لطلاب ومطوري علوم الحاسب من أجل:

- 🚀 **عرض** مشاريع التخرج وأفكار الشركات الناشئة.
- 🔍 **اكتشاف** مشاريع مبتكرة من مطورين حول العالم.
- 🤝 **البحث** عن متعاونين ومساهمين مفتوحي المصدر.
- 🏆 **المنافسة** في قائمة الصدارة العالمية.

## التقنيات المستخدمة (Tech Stack)

> تم بناء هذا المشروع باستخدام تقنيات الويب الحديثة. يُنصح بالإلمام بالتقنيات التالية قبل المساهمة في الكود.

| التقنية | الهدف أو الاستخدام |
|---|---|
| `Next.js 15` | App Router, Server Components |
| `React 19` | إطار عمل واجهة المستخدم (UI) |
| `Tailwind CSS` | تنسيق وتصميم الواجهات |
| `Sanity CMS` | إدارة المحتوى وقواعد البيانات |
| `NextAuth.js` | المصادقة وتسجيل الدخول |
