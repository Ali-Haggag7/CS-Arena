# مقدمة

أهلاً بيك في توثيق **CS-Arena**. الدليل ده هيغطي كل حاجة محتاج تعرفها — سواء كنت بتعرض مشروعك، أو بتساهم في الكود.

## ما هي CS-Arena؟

CS-Arena منصة مصممة لطلاب ومطوري علوم الحاسب عشان:

- 🚀 **يعرضوا** مشاريع التخرج وأفكار الـ Startups.
- 🔍 **يكتشفوا** مشاريع مبتكرة من مطورين حوالين العالم.
- 🤝 **يلاقوا** متعاونين ومساهمين Open Source.
- 🏆 **يتنافسوا** في قائمة الصدارة العالمية.

## التقنيات المستخدمة (Tech Stack)

> المشروع اتبنى بأحدث تقنيات الويب. يُنصح تكون عارف التقنيات دي قبل ما تساهم في الكود.

| التقنية | الاستخدام |
|---|---|
| `Next.js 15` | App Router, Server Components |
| `React 19` | إطار عمل واجهة المستخدم |
| `Tailwind CSS` | تنسيق وتصميم الواجهات |
| `Sanity CMS` | إدارة المحتوى وقواعد البيانات |
| `NextAuth.js` | المصادقة وتسجيل الدخول |

## إعداد سريع (Quick Setup)
```bash
git clone https://github.com/Ali-Haggag7/CS-Arena.git
cd CS-Arena
npm install
npm run dev
```
